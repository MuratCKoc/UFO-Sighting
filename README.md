## UFO-Sighting
---
The planet Earth needs to know what we have found!

#### Functionality
---
The user can to set multiple filters and search for UFO sightings using the following criteria based on the table columns:

  1. `date/time`
  2. `city`
  3. `state`
  4. `country`
  5. `shape`

#### Technology
---
* HTML 5
* Javascript
* CSS 3
* Bootstrap
